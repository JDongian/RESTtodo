/api/schedule/addclass

Adds a class to the PostgreSQL database. Checks user login state.

/api/schedule/delete

Takes in a POST verb referencing a specific schedule item, and deletes it. Not functional in the web client.

/api/login

A user endpoint, validates user login and produces a session token.

/api/validate

A user endpoint, validates session tokens stored in cookies.

/api/adduser

A user endpoint, adds a user with state 1. State is set to 0 when email confirmed.

/api/confirm

A user endpoint, handles email confirmations and sets user state to 0 if a valid confirmation link is provided.
